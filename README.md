# E-Commerce Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on an E-Commerce dataset using Python. The main objective of this analysis is to identify patterns, trends, customer behavior, sales performance, and business insights from raw data.

The project demonstrates practical skills in:
- Data Cleaning
- Data Visualization
- Statistical Analysis
- Outlier Detection
- Correlation Analysis
- Business Insight Generation

---

# 🚀 Features

✔ Data Cleaning & Preprocessing  
✔ Missing Value Handling  
✔ Duplicate Value Removal  
✔ Descriptive Statistical Analysis  
✔ Product Performance Analysis  
✔ Payment Method Analysis  
✔ Monthly Sales Trend Analysis  
✔ Distribution Analysis
